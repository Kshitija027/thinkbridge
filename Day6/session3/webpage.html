HTML Tags & Attributes

HTML- HTML (HyperText Markup Language) is the standard language for creating webpages. it tells the browser how to structure the content (text, images, links, videos, forms, etc.). HTML uses tags written inside < > to define elements.

  • <p> — Paragraph text
    Example: <p>Hello World</p>
    Output:

    Hello World

  • <a href> — Hyperlink
    Example: <a href="https://www.google.com">Google</a>
    Output: Google
  • <img src alt> — Image
    Example: <img src="https://via.placeholder.com/100" alt="Sample">
    Output:
  • <ul> <li> — List
    Example: <ul><li>Item 1</li><li>Item 2</li></ul>
    Output:
    • Item 1
    • Item 2
  • <div id class style> — Container
    Example: <div id="box" class="container" style="border:1px solid #000;">Content</div>
    Output:
    Content
  • <form action method input> — Form
    Example: <form action="#" method="post"><input type="text" placeholder="Name"></form>
    Output:
  • <h1>-<h6> — Headings
    Example: <h2>Heading 2</h2>
    Output:

    Heading 2

  • <table> <tr> <td> — Table
    Example: <table border="1"><tr><td>A</td><td>B</td></tr></table>
    Output:
    AB

css properties

CSS (Cascading Style Sheets) is used to style and design web pages. It controls colors, fonts, layouts, spacing, backgrounds, animations, and much more. HTML gives the structure, but CSS makes it look beautiful.

  • color — Text color
    Red text
  • background-color — Background
    Yellow box
  • margin — Outer spacing
    Margin box
  • padding — Inner spacing
    Padding box
  • border — Element border
    Blue border
  • font-size — Text size
    Medium text
  • display — Element display
    Inline-block box
  • width / height — Element size
    Box size
  • text-align — Align text
    Centered text

javascript properties

JavaScript (JS) is a programming language that makes web pages interactive and dynamic. It runs inside the browser and can also run on servers (with Node.js).With JavaScript, you can: Change HTML & CSS on the flyRespond to user actions (click, input, scroll, etc.)Validate forms
